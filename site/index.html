Install Guide

四步安装，开始使用

主题包遵循统一目录约定，解压后刷新 Yulora 主题列表即可切换。

01

打开主题目录

在 Yulora 设置页点击「打开主题目录」，定位到 <userData>/themes/。

02

下载并解压

从 GitHub Releases 下载目标主题 zip，解压到主题目录。

03

确认目录结构

确认解压后形态为 <themeId>/manifest.json，不要多嵌或散放。

04

刷新并切换

返回设置页点击「刷新主题」，然后在下拉列表中切换到目标主题。

!

新增主题展示时，推荐同时补充 showcase.json 与固定命名封面图 cover.png。如果不提供封面图，主页会自动使用颜色和参数信息生成展示卡片。
