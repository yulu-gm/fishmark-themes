Yulora Themes

把主题下载入口、展示页和发布链路放到同一个仓库里。

这里集中提供 Yulora 官方主题包的浏览、下载与安装说明。当前收录三套主题： Ember Ascend、Pearl Drift、 Rain Glass。

下载最新主题包 查看 GitHub 仓库

安装方式

  1. 打开 Yulora 主题目录
  2. 下载 zip
  3. 解压到 `themes/`
  4. 回到应用点击“刷新主题”

压缩包解压后必须保留 `/manifest.json` 这一层目录结构。

Theme Library

三套主题，三种完全不同的氛围。

第一版页面先使用氛围卡与参数摘要，不强绑定真实应用截图，这样主题可以更快发布和迭代。

Install Guide

用户如何安装

  1. 在 Yulora 设置页点击“打开主题目录”。
  2. 下载目标主题 zip 并解压到 `/themes/`。
  3. 确认解压后目录形态为 `/manifest.json`。
  4. 返回设置页点击“刷新主题”，然后切换到目标主题。

Package Contract

主题包结构约束

pearl-drift/
  manifest.json
  tokens/
  styles/
  shaders/

不要把 `manifest.json` 直接散在 zip 根目录，也不要多嵌一层随机文件夹名。Yulora 只会扫描 `/themes/<id>/manifest.json`。

Yulora Themes 由 Yulora 项目维护。
